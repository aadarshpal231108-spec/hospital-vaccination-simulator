# Hospital Vaccination Impact Simulation

An interactive Streamlit dashboard that simulates infection spread in a hospital or patient population using an SIR-inspired model with vaccination and intervention strategy controls.

## Features
- Susceptible, Vaccinated, Infected, and Recovered state tracking
- Intervention strategy with configurable vaccination campaign start day
- Herd immunity threshold analysis
- Multiple contact network patterns
- Final network visualization
- Interactive parameter tuning from the sidebar

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
